### Installation
```bash
$ git clone https://github.com/KIMHT98/AlcoholHeaven.git
$ cd AlcoholHeaven
```
### BackEnd
```bash
$ cd AlcoholHeaven
$ mvn clean install
$ mvn spring-boot:run
```

### FrontEnd
```bash
$ cd alcohol-heaven
$ npm install
$ npm run dev
```
